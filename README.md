# MoonCompass — showcase

A single-page showcase for **MoonCompass** (project codename) — an invitation to look up.
The app brings the quiet pleasure of watching the sky to ordinary people: know when the moon
passes by your window, be there for the sunrise and sunset, and aim your phone to find the
planets, the moon, even passing comets.

Project in development · expected end of summer 2026.

This repo is the public marketing page only. The application source is private.

## Local preview
Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```
